CSS anidado, Variables y Funciones

Esta página demuestra tres conceptos: CSS anidado, CSS variables, y CSS funciones. Use los controles a continuación para experimentar con los colores.

Controles interactivos

Anidado

Esta tarjeta usa reglas anidadas. Adentro de esta .card, los estilos para h3, p, ul, y .badge están anidados dentro del bloque de la tarjeta.

  • Las reglas permanecen agrupadas.
  • La estructura es más fácil de leer.
  • La relación entre padre e hijo es más clara.
Ejemplo de anidamiento

Variables

Los colores de la página dependen de variables como --primary, --bg, y --text. Cambiar una variable actualiza muchas partes del diseño al mismo tiempo.

  • Cambios globales de color sencillos
  • Más consistencia a través de los componentes
  • Menos repetición en CSS
Propiedades personalizadas

Funciones

Esta página usa funciones CSS como var(), hsl(), calc(), min(), y clamp().

  • var() lee las variables CSS
  • hsl() genera colores
  • calc() calcula el espaciado
  • clamp() hace que el texto sea responsive
CSS dinamico

Cambio de color en vivo

Mueva los controles deslizantes de arriba. JavaScript solo actualiza los valores de las variables CSS, mientras que CSS maneja los cambios visuales.

Esto es util porque mantiene la lógica de estilo dentro de CSS, lo que hace que el código sea más limpio y fácil de mantener.
